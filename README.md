# Student Gradebook Assignment

## Instructions
1.  Clone this repository.
2.  Open the project in your IDE (IntelliJ, Eclipse, VS Code).
3.  Navigate to `src/main/java/edu/vwcc/grades`.
4.  Implement the `TODO` items in `Student.java`, `Gradebook.java`, and `DataLoader.java`.
5.  Run the tests locally using `mvn test` or your IDE's test runner to verify your work.
6.  Commit and push your changes to GitHub.

## Autograding
This repository is configured with GitHub Actions. When you push your code, the autograder will run the tests defined in `src/test/java`.
*   **Green Check**: All tests passed.
*   **Red X**: One or more tests failed. check the "Actions" tab for details.

## Requirements
*   **Java 21**
*   **Maven**
